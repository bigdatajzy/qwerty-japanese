# 新功能需求建议

> 版本：1.1 | 日期：2026-03-17 | 状态：待评审

---

## 需求列表

### 🔴 高优先级 (P0)

#### 1. 打字音效系统
**描述**：输入正确/错误时的音效反馈，增强打字节奏感

| 项目 | 内容 |
|------|------|
| 正确音效 | 清脆的 "滴" 声，类似按键音 |
| 错误音效 | 低沉的 "嗡" 声 |
| 完成音效 | 练习完成时的提示音 |
| 音量控制 | 支持 0-100% 音量调节 |
| 音效开关 | 可单独开启/关闭各类音效 |

**技术实现**：使用 Web Audio API，音效文件放 `public/sounds/`

---

#### 2. 盲打训练模式
**描述**：不显示假名/汉字，仅显示罗马字提示，训练真正的盲打能力

| 项目 | 内容 |
|------|------|
| 隐藏假名 | 仅显示当前待输入的罗马字 |
| 罗马字提示 | 可选择显示/隐藏下一个词的罗马字 |
| 难度分级 | L1:显示罗马字 → L2:隐藏罗马字 → L3:无任何提示 |

---

#### 3. 练习历史记录
**描述**：保存每次练习的详细记录，支持回顾和分析

| 项目 | 内容 |
|------|------|
| 记录内容 | 练习时间、词库、WPM、正确率、错误详情 |
| 存储方式 | localStorage |
| 历史展示 | 最近 50 条记录 |
| 数据导出 | 支持导出为 JSON |

---

#### 4. 日语单词练习 (新增)
**描述**：添加 JLPT 词汇练习，补全「假名→句子」之间的单词环节

| 项目 | 内容 |
|------|------|
| 词库内容 | 假名 → 汉字 → 释义 → 罗马字 |
| 显示模式 | 可切换：仅假名 / 假名+汉字 / 假名+释义 |
| 练习方向 | 看假名打汉字 / 看汉字打假名 / 看释义打假名 |

**词库规划**：

| 词库 | 词汇量 | 难度 | 状态 |
|------|--------|------|------|
| N5 词汇 | ~800 词 | 入门 | 规划 |
| N4 词汇 | ~1200 词 | 基础 | 规划 |
| N3 词汇 | ~1500 词 | 中级 | 规划 |
| N2 词汇 | ~2000 词 | 中高级 | 规划 |
| N1 词汇 | ~2500 词 | 高级 | 规划 |

**数据来源**：可参考 [kanjiapi.dev](https://kanjiapi.dev/) 或开源 JLPT 词库

---

### 🟡 中优先级 (P1)

#### 5. 听写练习模式
**描述**：播放假名/单词发音，用户输入对应假名

| 项目 | 内容 |
|------|------|
| 发音来源 | 浏览器 Speech Synthesis API |
| 播放控制 | 播放/暂停/重播 |
| 难度选择 | 单假名 → 单词 → 短句 |
| 听写内容 | 假名听写 / 单词听写 |

---

#### 6. 错题本功能
**描述**：记录练习中的错误，重点复习

| 项目 | 内容 |
|------|------|
| 自动收录 | 错误率 >20% 的词自动加入错题本 |
| 错题练习 | 专门针对错题进行强化训练 |
| 手动添加 | 用户可手动添加难记的词 |
| 清除记录 | 支持删除单条/清空全部 |

---

#### 7. 学习路径/课程
**描述**：规划从零到 N1 的完整学习路线

| 项目 | 内容 |
|------|------|
| 阶段划分 | 假名 → 单词 → 短句 → 文章 |
| 阶段目标 | 每个阶段设置明确的学习目标 |
| 进度可视化 | 整体进度百分比 + 各阶段完成度 |
| 推荐练习 | 根据当前水平推荐下一步练习 |

---

#### 8. 进度追踪系统
**描述**：记录用户在每个词库的练习进度

| 项目 | 内容 |
|------|------|
| 进度保存 | 已练习词数/总词数 |
| 掌握度标记 | 正确率 >95% 标记为已掌握 |
| 复习提醒 | 已掌握词汇定期复习提醒 |
| 可视化 | 进度条 + 饼图展示 |

---

### 🟢 低优先级 (P2)

#### 9. 自定义词库
**描述**：用户可添加自己的词库

| 项目 | 内容 |
|------|------|
| 添加方式 | 手动输入 / CSV 导入 / JSON 导入 |
| 词库管理 | 编辑 / 删除 / 分享 |
| 云同步 | (可选) 账户登录后同步 |

---

#### 10. 键盘快捷键
**描述**：常用操作的键盘快捷键

| 快捷键 | 操作 |
|--------|------|
| Space | 提交输入/下一题 |
| Enter | 确认/继续 |
| Tab | 切换下一个输入 |
| Esc | 退出练习 |
| Ctrl+R | 重新开始 |

---

#### 11. 成就系统
**描述**：完成特定里程碑解锁成就

| 成就 | 条件 |
|------|------|
| 初学者 | 完成第一次练习 |
| 假名大师 | 完成所有假名练习 |
| 单词王者 | 完成 N5/N4 词汇练习 |
| 速度王者 | WPM 超过 60 |
| 正确率 100% | 一次练习无错误 |

---

#### 12. 每日挑战
**描述**：每日更新的练习挑战

| 项目 | 内容 |
|------|------|
| 每日任务 | 固定的练习目标 |
| 排行榜 | (可选) 社区排名 |
| 奖励 | 连续打卡解锁成就 |
