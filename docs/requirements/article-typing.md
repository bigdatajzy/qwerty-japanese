# 新需求：日语文章打字练习

## 需求概述

新增日语文章打字练习模块，支持预制文章和自定义文章上传练习。

## 功能列表

### 1. 预制文章
- 提供几篇经典的日语短文（如日语能力考试阅读素材）
- 按难度分类（入门、初级、中级）
- 支持注音显示（见下文）

### 2. 自定义文章上传
支持多种格式：

**格式 A：纯文本 (.txt)**
- 每一行是一个独立的待输入内容
- 空行自动跳过

**格式 B：JSON 格式（简单）**
```json
{
  "title": "文章标题",
  "content": "完整的日语文章内容..."
}
```

**格式 C：带注音的 JSON（高级）**
```json
{
  "title": "标题",
  "content": [
    {
      "text": "日本語を勉強するのは楽しいです",
      "ruby": [
        {"word": "日本語", "kana": "にほんご"},
        {"word": "勉強", "kana": "べんきょう"}
      ]
    }
  ]
}
```

**格式 D：自动注音**
```json
{
  "title": "标题",
  "content": "日本語を勉強するのは楽しいです",
  "autoRuby": true
}
```

### 3. 文章练习界面
- 显示文章内容（待输入）
- 实时高亮当前输入位置
- 显示进度和正确率
- **支持注音显示**（在汉字上方显示小字号平假名）
