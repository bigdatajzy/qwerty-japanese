# 日语打字练习网站 - 详细需求文档 (PRD)

## 1. 项目概述

**项目名称**：Qwerty Japanese (日语打字练习)

**项目目标**：为日语学习者提供打字练习工具，通过键盘输入罗马字转换为假名/汉字，锻炼肌肉记忆

**对标参考**：[Qwerty Learner (qwerty.kaiyi.cool)](https://qwerty.kaiyi.cool)

**技术栈**：
- React + TypeScript + Vite
- Tailwind CSS
- 部署：Vercel / Cloudflare Pages

---

## 2. 核心功能

### 2.1 打字练习界面

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 单词/假名显示 | 显示待输入的假名/汉字/罗马字 | P0 |
| 输入框 | 用户输入罗马字 | P0 |
| 实时反馈 | 输入正确：绿色高亮 + 音效<br>输入错误：红色闪烁 + 提示正确值 | P0 |
| 进度条 | 显示当前章节进度 | P1 |
| 下一个提示 | 预览下一个待输入内容 | P1 |

### 2.2 统计功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| WPM | 每分钟正确输入字符数 | P0 |
| 正确率 | 正确字符数 / 总字符数 | P0 |
| 速度曲线 | 实时显示输入速度变化 | P1 |
| 历史记录 | 保存每次练习的统计 | P2 |

### 2.3 词库管理

| 词库 | 描述 | 优先级 |
|------|------|--------|
| 平假名 | あいうえお... 基础练习 | P0 |
| 片假名 | アイウエオ... 基础练习 | P0 |
| 浊音/半浊音 | がぎぐげご... ばびぶべぼ... | P0 |
| 拗音 | きゃしゅちょ... | P0 |
| N5 词汇 | JLPT N5 级别单词 | P1 |
| N4 词汇 | JLPT N4 级别单词 | P2 |
| N3 词汇 | JLPT N3 级别单词 | P2 |
| 常用汉字 | 常见汉字 + 读音 | P2 |

### 2.4 练习模式

| 模式 | 描述 | 优先级 |
|------|------|--------|
| 顺序模式 | 按词库顺序依次练习 | P0 |
| 随机模式 | 随机抽取词库内容 | P0 |
| 盲打模式 | 不显示罗马字提示 | P1 |
| 听写模式 | 播放发音，输入对应假名 | P2 |

### 2.5 用户体验

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 发音播放 | 点击假名播放读音 | P1 |
| 音效反馈 | 输入正确/错误时的音效 | P1 |
| 暗色主题 | 深色界面 | P1 |
| 章节选择 | 选择要练习的章节 | P0 |
| 键盘指引 | 显示键盘按键位置 | P2 |

## 4. 页面结构

### 4.1 首页 (Home)
- 项目标题 + 副标题
- 开始练习按钮
- 词库选择
- 显示统计摘要

### 4.2 练习页面 (Practice)
- 顶部：当前词库 / 章节名
- 中间：大号假名/汉字显示
- 下方：输入框 + 下一个预览
- 底部：统计栏 (WPM / 正确率 / 进度)
- 侧边：设置面板（音效、主题等）

### 4.3 结果页面 (Results)
- 本次练习统计
- 错误列表
- 重新练习按钮
- 返回首页

### 4.4 词库管理 (Dicts)
- 词库列表展示
- 添加自定义词库（可选）
- 词库搜索
