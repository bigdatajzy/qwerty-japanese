# 日语单词练习需求

> 版本：1.0 | 日期：2026-03-18 | 状态：开发中

---

## 需求概述

添加 JLPT 词汇练习，补全「假名→句子」之间的单词环节

## 功能详情

### 词库内容
- 假名 → 汉字 → 释义 → 罗马字

### 显示模式（可切换）
- 仅假名
- 假名 + 汉字
- 假名 + 释义

### 练习方向
- 看假名打汉字
- 看汉字打假名
- 看释义打假名

## 词库规划

| 词库 | 词汇量 | 难度 | 状态 |
|------|--------|------|------|
| N5 词汇 | ~800 词 | 入门 | 开发中 |
| N4 词汇 | ~1200 词 | 基础 | 规划 |
| N3 词汇 | ~1500 词 | 中级 | 规划 |
| N2 词汇 | ~2000 词 | 中高级 | 规划 |
| N1 词汇 | ~2500 词 | 高级 | 规划 |

## 技术实现

1. 创建 `public/words/n5.json` 词库文件
2. 创建 `src/views/WordsView.vue` 单词列表页
3. 创建 `src/views/WordPracticeView.vue` 单词练习页
4. 添加路由配置
5. 在首页添加入口

## 数据格式

```json
{
  "n5": [
    {
      "id": "1",
      "hiragana": "にほんご",
      "kanji": "日本語",
      "meaning": "日语",
      "romaji": "nihongo"
    }
  ]
}
```
